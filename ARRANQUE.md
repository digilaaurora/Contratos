# Arranque de la app Contratos LA

## Requisitos previos
- Node.js instalado (para usar `npx`)

## Cómo arrancar

Abre una terminal en la carpeta del proyecto y ejecuta:

```
npx serve . -p 3000
```

Luego abre el navegador en:

```
http://localhost:3000
```

## Notas
- El servidor debe estar corriendo para que la app funcione.
- No cerrar la terminal mientras se usa la app.
- Si el puerto 3000 está ocupado, cambia el número: `npx serve . -p 3001`
